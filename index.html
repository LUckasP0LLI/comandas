Configure seu banco de dados Firebase para sincronização em tempo real entre todos os dispositivos.
Como criar (gratuito):
1. Acesse console.firebase.google.com
2. Criar projeto → dê um nome → continuar
3. Na tela do projeto: clique em </> → registrar app
4. Copie os valores do firebaseConfig acima
5. Menu lateral: Build → Realtime Database → Criar banco
6. Escolha "Iniciar no modo teste" → criar
7. Cole os dados e clique em Salvar
